# Xpoz Research Report

## Overview

Xpoz (xpoz.ai) is an Israeli startup founded in 2023 in Herzliya that has repositioned itself from a disinformation-detection platform into "the social data layer for AI." The core product is an MCP (Model Context Protocol) server that enables AI agents — Claude, ChatGPT, and others — to query Twitter/X, Instagram, Reddit, and TikTok data through natural language, without requiring platform API keys or dealing with rate limits. The company claims 1.5B+ indexed social media posts.

Led by CEO Ran Farhi, COO Mohar Bar, co-founder Danielle Israel, and VP Engineering Shy Engelberg, the team has backgrounds in Israeli cybersecurity and defense tech. The company is pre-funding with 11-50 employees and has been rapidly expanding its developer ecosystem since January 2026, shipping 6 GitHub repos (MCP server, Python SDK, TypeScript SDK, agent skills, Claude Code plugins, and a lead-gen workflow example) in under two months.

The pricing is aggressive: a free tier with 100K results/month, Pro at $20/month for 1M results, and Max at $200/month for 10M results — dramatically undercutting the official Twitter/X API ($100-$42,000+/month) and traditional social listening tools like Brandwatch ($60K+/year). However, the product is very new, community traction is minimal, and key questions about data sourcing, legal compliance, and long-term sustainability remain unanswered.
